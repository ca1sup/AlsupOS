1. Start a New Task (The "Branch")

When you sit down to code a new feature (like fixing a bug or adding a page):

Click main in the bottom-left corner of VS Code.

Select + Create new branch...

Name it something specific (e.g., update-login-page).

2. Save Your Work (The "Commit")

After you make edits:

Go to the Source Control sidebar.

Stage your changes (Click +).

Write a message and click Commit.

Click Publish Branch (Sync).

3. Merge It (The "Pull Request")

When you are done and want the code in the official main version:

Go to your GitHub repository in your browser.

Click the green Compare & pull request button.

Click Merge pull request.
